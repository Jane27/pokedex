pokedex is a sample react project implemented by React.

## Libraries used:
- React router
- Rematch
- Styled-component
- Lodash

## Api used:
- https://pokeapi.co/api/v2/pokemon

## How to run it locally

Prerequisite:

### 1. Install npm packages
```
npm install
```

### 2. Start locally
```
npm run start
```

## Build for production

```
npm run build
```
